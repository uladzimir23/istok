# 50 — Roadmap

Фазовое планирование проекта. Каждая фаза — отдельный документ `Phase N — <Title>.md`.

## Текущие фазы (предварительно)

- **Phase 0 — SDD-каркас** (текущая, 2026-05-01) — `CLAUDE.md`, скиллы, агенты,
  docs/, бриф, аудит. Без кода.
- **Phase 1 — Architecture decisions** — ADR-001..ADR-008 фиксируют стек.
- **Phase 2 — Foundation** — `pnpm`/`bun` + Next.js + PocketBase локально, схема
  коллекций, базовый layout.
- **Phase 3 — Каталог** — карточки товаров (Элис, кресла, корпусная), фильтры, заявка.
- **Phase 4 — Маркетинг-инфраструктура** — Метрика + GA4, формы → Telegram, schema.org,
  OG, sitemap.
- **Phase 5 — Cutover** — staging, тесты, переключение DNS, редиректы со старых URL,
  убрать Tilda.
- **Phase 6 — Post-launch** — мониторинг, оптимизация, A/B.

Конкретика — после Phase 1 (стек зафиксирован → можно оценивать сроки).
