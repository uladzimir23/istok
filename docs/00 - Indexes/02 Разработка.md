## Зоны кода (Phase 1 имплементация)

```
istok/
├── web/                           # Next.js 15 (App Router, TypeScript strict)
│   └── src/
│       ├── app/                   # роутинг: страницы, layouts, route handlers
│       │   ├── (site)/            # публичная витрина
│       │   └── krovatki/          # раздел ELIS с визуальной темой
│       └── widgets/               # FSD-lite: виджеты, переиспользуемые блоки
├── content/                       # контент-as-code (MDX + TypeScript)
│   ├── schema.ts                  # Zod-схемы для всех типов продуктов
│   └── products/                  # MDX-файлы товаров (кресла, кроватки, корпус)
├── nginx/                         # конфиг nginx для Docker deploy
├── docker-compose.yml             # web + nginx; Phase 2 добавит pocketbase-сервис
├── .github/
│   └── workflows/                 # GH Actions: typecheck + lint + build + deploy
└── scripts/                       # check-secrets.sh, generate-og и пр.
```

(Структура зафиксирована ADR-007. PocketBase появится в Phase 2 как отдельный сервис
в compose, без monorepo-изменений.)

## Риски и блокеры

🟡 **VPS provisioning не сделан.** Нужно выбрать провайдера (Aeza или Hetzner Helsinki),
поднять сервер, настроить Docker + nginx + certbot. До этого staging недоступен.

🟡 **Контент детских кроваток нужно перенести из PDF в MDX.** Источник — 21-страничный
PDF-каталог «Элис» (`Фотографии кроваток 2/Обновленный каталог.2.pdf`). Вручную или
через скрипт-парсер.

🟡 **Токены палитры ELIS не финализированы.** Для cascade-layer-темы раздела `/krovatki`
нужны финальные цветовые токены бренда «Элис». Пока не определены — блокируют верстку
темы.

🟡 **Контент-зависимость от клиента.** Без цен и каталога кресел сайт не запустится
полноценно. Параллельно с разработкой — бить на доставке контента.

🟡 **AI-пайплайн не формализован.** У клиента есть рендеры из Gemini/Krea, но без
пресетов и системы. Нужен playbook.

🟡 **Tilda как «работающий блокер».** Cutover требует подготовки DNS, тестов, окна
простоя — отдельная фаза с чек-листом.
