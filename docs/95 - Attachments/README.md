# 95 — Attachments

Исходные материалы клиента, на которые ссылаются документы. Тяжёлые бинарные файлы
(PDF, фото, рендеры) живут здесь.

## Что сюда пойдёт (после переноса из корня папки)

- `elis-catalog.pdf` — актуальный 21-страничный PDF-каталог линейки «Элис».
- `elis-catalog-pages/` — каждая страница каталога как отдельный PDF (для рассылок).
- `elis-photos/<sku>/` — структурированные фото по 8 моделям «Элис».
- `raw/ai-renders/` — AI-рендеры (Gemini/Krea), эксперименты клиента.

## На момент init

Файлы пока лежат в корне папки и в исходных папках («Мебель/», «Фотографии кроваток 2/»),
**не в git** (см. `.gitignore`). Перенос — отдельная задача, требует переименования
файлов в латиницу и структурирования по SKU.
