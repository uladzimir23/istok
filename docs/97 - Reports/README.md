# 97 — Reports

Отчёты для клиента (фабрики). По итогам фаз, аудитов, рекламных кампаний (когда будут).

## Что сюда пойдёт

- Отчёт по итогам Phase 0 (SDD-каркас) — для клиента в простой форме «что мы сделали
  и зачем».
- Ежемесячные отчёты по аналитике (после запуска Метрики).
- Отчёты по A/B-тестам.
- Отчёты по SEO-позициям.
