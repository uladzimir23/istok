# 60 — Content

Контент для сайта: тексты главной, описания моделей, маркетинговые блоки, FAQ.

## Что сюда пойдёт

- `elis/<sku>.md` — описание каждой модели «Элис» (8 файлов).
- `chairs/<model>.md` — описания театральных кресел.
- `office/<category>.md` — описания корпусной мебели по категориям.
- `home.md` — тексты главной.
- `about.md` — текст «О компании».
- `goszakaz.md` — текст лендинга «Госзаказы».
- `faq.md` — частые вопросы.
- `services.md` — доставка, сборка, гарантия.

Когда появится PocketBase, эти тексты переедут в БД через сидинг-скрипт. Здесь они
живут как **first draft / source of truth для копирайта**.
