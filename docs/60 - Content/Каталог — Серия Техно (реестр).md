# Каталог — Серия Техно (полный реестр)

## 0. Источник данных

Документ построен по 5 PNG-скринам официального каталога серии Техно
(`~/Documents/zavody-rb/istok/Мебель/`), полученным от заказчика.

> фабрики. Текущие 12 placeholder-карточек в `content/products/cabinets/`

## 5. Что меняется в архитектуре сайта

### 5.1 Схема `Product`

Добавляем поля:

```ts
series: z.enum(["tehno"]).optional(),       // когда появятся другие — расширим
subCategory: z.string().optional(),         // stellazhi | shkafy | stoly | komody | tahty | polki | tumby
orientation: z.enum(["R", "L"]).optional(), // для тахт
```

### 5.2 Удаление placeholder'ов

Текущие 12 файлов в `content/products/cabinets/` (`odnodverny.mdx`,
`s-dvercey-i-4-yaschikami.mdx`, etc.) — **удаляем**. Их фото с Tilda
сохраняем в `_archive/tilda-cabinet-photos/` на случай если что-то
из них совпадёт с реальной серией.

### 5.3 Замена навигации

| Сейчас | Будет |
| --- | --- |
| `/komody/` `/stoly/` `/stelazhi/` `/shkafy/` (4 заглушки) | `/korpusnaya/tehno/{stellazhi,shkafy,stoly,komody,tahty,polki,tumby}/` (7 разделов) |
| `CollectionSwitcher` для корпусной с 4 пустыми табами | переключатель серий (пока 1) → подкатегории Техно |

### 5.4 CategoriesBoard (на главной)

Сейчас 4 квадрата (Кресла / Корпусная / Кроватки / Проекты). Корпусная
ведёт на `/korpusnaya/`. Это остаётся, но внутри теперь Серия Техно.
