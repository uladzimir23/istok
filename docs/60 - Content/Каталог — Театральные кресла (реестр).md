| **Боковина кресла** | твёрдая древесина с шиповым соединением, втулки для осей |
| **Стойка кресла** | металлическая, опорная |

### 3.2 Сборка в секции

> Кресла собираются в секции, в зависимости от планировки зала. Крепление в ряд через **болты, шпильки, гайки М10**. Изготавливаются:
> - **с креплением к полу** — 2 дюбеля 8×80 мм на каждую стойку
> - **без крепления** (переносные)

Это важно для тендеров — фабрика готова под любую конфигурацию.

### 3.3 Договорные условия (универсально)

> Цвет деревянных, металлических и тканевых материалов, а также количество кресел в секции, определяется заказчиком при заключении договора.

> Гарантия на поставленные кресла — **24 месяца** с момента ввода в эксплуатацию.

## 4. Что отсутствует (запрос клиенту)

### 4.1 Размеры — HIGH

- [ ] **М2**: Г и Ш (Tilda пусто — нужны)
- [ ] **М3-1**: В, Г, Ш (Tilda пусто)
- [ ] **Высота сиденья** — рабочая (от пола до края подушки) — критично для проектировщиков залов
- [ ] **Высота откидного сиденья сложенного** — для проверки прохода в ряду
- [ ] **Расстояние между креслами в секции** — стандарт / опция

### 4.2 Материалы (детально) — HIGH

- [ ] **Каркас**: какой металл (сталь / алюминий?), толщина профиля
- [ ] **Сиденье**: фанера (берёза?), толщина слоёв
- [ ] **Поролон**: плотность (кг/м³), толщина в сиденье и спинке
- [ ] **Обивка**: тип ткани (мебельная букле / экокожа / велюр?), плотность (г/м²), огнестойкость по СТБ
- [ ] **Откидной механизм**: производитель (Эконом / Партнёр / импорт?), ресурс циклов
- [ ] **Боковина**: порода древесины (бук / дуб / берёза?), сорт

### 4.3 Палитра — HIGH

- [ ] **Цвета ткани** — палитра по моделям (М1-ПМ-1 могут отличаться от М3-серии)
- [ ] **Цвета каркаса** — стандарт чёрный? + RAL опционально?
- [ ] **Цвета деревянных боковин** — лак прозрачный / морилка / краска?
- [ ] **Цены при выборе нестандартного цвета** — наценка?

### 4.4 Опции комплектации — HIGH

- [ ] **Нумерация ряда/места**: металлическая табличка / пластик / гравировка — варианты + цены
- [ ] **Логотип на спинке** (для залов с брендингом) — цена
- [ ] **Подлокотник усиленный** (для М-серии) — опция?
- [ ] **Чехол на кресло** (для нерабочего сезона) — производит фабрика?
- [ ] **Кронштейны для подстаканника / письменного столика** — для ВУЗ-аудиторий

### 4.5 Коммерческие условия

- [ ] **МОЗ** — минимальный заказ серии (10 шт? 50?)
- [ ] **Срок производства** — серия 100 кресел / серия 500 кресел
- [ ] **Цена в долларах / евро / российских рублях** — для экспорта в РФ / СНГ
- [ ] **Условия для тендеров** — оплата по факту, банковская гарантия?

### 4.6 Фото / визуал — MED

- [ ] **Hi-res рендеры** (3D-модели) — для главного фото на сайте
- [ ] **Фото в зале** (реальная установка) — для портфолио / hero-секций
- [ ] **Чертежи в DWG / PDF** — для архитекторов / проектировщиков
- [ ] **Видео-демонстрация откидного механизма** — 5-10 сек
