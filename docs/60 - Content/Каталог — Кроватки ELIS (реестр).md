## 3. Общие свойства всех моделей

### 3.1 Материалы (база)

| Материал | Применение |
| --- | --- |
| Натуральная **берёза** | Каркас, ламели, ножки |
| **MDF** | Плоские элементы (спинка, бортики) — все модели кроме ЭЛИС-1 |
| **Гипоаллергенное лакокрасочное покрытие** | Все модели — детский экологичный лак |
| **Обработка маслом** | ЭЛИС-4/5/6 — накладки на спинке |

### 3.2 Цветовая палитра

**Что мы видим в PDF-фото:**
- Белый (основной)
- Молочно-белый
- Деревянные элементы — **берёза натуральная** (светлый тон)

**Что нужно подтвердить у клиента:**
- [ ] Сколько вариантов покраски доступно — только белый или есть цветные (мятный, розовый, серый из фотографий?)
- [ ] Сэмплы / RAL-номера / hex'ы для дополнительных цветов
- [ ] Цены — отличаются ли цены по цвету?

> На фото в каталоге видны акцентные цвета у Эльзы (мятный фон), Доминики (розовый), но это **декор комнаты**, не самой кроватки. Сами кроватки во всех фото — **белые**.

### 3.3 Сборка и доставка

PDF не уточняет — нужно подтвердить:
- [ ] Кровать поставляется в собранном или разобранном виде?
