# 67 — SMM

Социальные сети и контент-маркетинг. Заполняется после Phase 4, когда есть сайт и
аналитика, и понятно, что промотим.

## У клиента есть сейчас

- Instagram: @istok_etg (активность не аудирована, нужно посмотреть).
- WhatsApp / Telegram / Viber как каналы прямой коммуникации с клиентами.

## Что сюда пойдёт

- Channels — список каналов и их роли.
- Content pillars — о чём говорим (производство, кейсы, новые модели, забота о детях,
  сертификации).
- Cadence — частота публикаций.
- Post templates — шаблоны под типы постов (карточка модели, кейс госзаказа,
  закулисье производства, отзыв клиента).
- Hashtags & captions — список тегов под BY-аудиторию.
- Visual grid — стилистика ленты Instagram.
- Metrics — что измеряем.
