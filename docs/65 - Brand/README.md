# 65 — Brand

Брендбук проекта. Закладывается в Phase 1 параллельно с архитектурой.

## Содержимое

- `assets/` — логотипы, шрифты, иконки в исходниках (SVG, woff2).
- (план) `Identity.md` — позиционирование, миссия, ценности.
- (план) `Visual language.md` — палитра, токены, типографика.
- (план) `Voice & tone.md` — стиль текстов для двух брендов (Исток B2B / ELIS B2C).
- (план) `Photography.md` — стилистика интерьерных фото для «Элис».
- (план) `Logo guidelines.md` — правила использования логотипа.

## Что у нас уже есть

- **Логотип в SVG** — клиент предоставил вектор `Компания Исток - Вектор.svg`.
  Будет перенесён в `assets/logo-istok-vector.svg`.
- **PDF-каталог «Элис»** — 21 страница, бренд ELIS-MEBEL Kids Beds. Палитра видна:
  кремовый/бежевый фон, тёплая фотография в детских интерьерах, тонкие шрифты для
  подписей, классическая типографика для названий.

## Что нужно от клиента

- Шрифты, если есть лицензированные (или замена через Google Fonts с похожей
  стилистикой).
- Точные значения цветов (HEX/RGB) — если есть гайдлайн.
- Тон-of-voice референсы (как обычно говорим с покупателями кроваток vs тендерными
  специалистами).
