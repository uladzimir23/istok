# ELIS-MEBEL — Visual language

> **TL;DR.** ELIS-MEBEL Kids Beds — нежный потребительский суббренд. Тёплая
> кремово-бежевая палитра, спокойные тона, детская айдентика без китча. Фон
> `#F1EDE0`, акцент `#DBD7CC`, текст `#5B5046` / контрастный `#271910`.
> Tilda использовала Montserrat + Comfortaa — оставляем как кандидатов на
> типографику до утверждения дизайнером.

## Источник

- 21-страничный PDF-каталог «Обновленный каталог.2.pdf» — основной источник палитры.
- Образцы пиксель-пик через `python3 + PIL`: cover (page 1) + content (page 3 «Николь»).
- Tilda HTML — источник списка шрифтов в текущем сайте.

> быть переименованы дизайнером при финализации.

### Семантика

- **Тёплая бежево-кремовая база** транслирует уют, природность, безопасность —
  ключевые ассоциации для родителей при выборе детской мебели.
- **Контраст низкий**, не агрессивный. Дети, спокойствие, сон — а не маркетплейсный
  «купи сейчас».
- **Нет ярких цветов в фирстиле.** Яркие акценты приходят только из фотографий
  интерьеров (мятный, пастельно-розовый, голубой) — это акценты сцен, не бренда.
  Не закладываем «фирменный мятный» — он будет конфликтовать с реальностью продаж.

### Применение в Web

```scss
@layer theme.brand-elis {
  :root {
    --color-bg-primary: #F1EDE0;
    --color-bg-secondary: #DBD7CC;
    --color-text-primary: #271910;
    --color-text-secondary: #5B5046;
    --color-text-muted: #877C73;
    --color-accent: #A19991;
  }
}
```

Раздел `/krovatki` (см. [[ADR-003]]) подключает класс `.theme-elis` на root-layout
страницы и переопределяет CSS custom properties.

## Типографика

### Текущее (Tilda)

В HTML-экспорте Tilda подключены два шрифта:
- **Montserrat** (weights 100–900, latin + cyrillic) — UI/основной текст.
- **Comfortaa** (weights 300–700, latin + cyrillic) — акцент/заголовки декоративные.

> Comfortaa в каталоге PDF не виден — там используется другой акцентный шрифт
> (тонкий serif с тонкими засечками для заголовков типа «"НИКОЛЬ" ЭЛИС-1»).
> Это расхождение между Tilda-сайтом и печатным каталогом.

### Кандидаты для нового сайта

> Утверждается дизайнером при финализации брендинга. Пока — рабочие предположения.

| Слой | Кандидат | Почему |
| --- | --- | --- |
| UI / body | **Montserrat Variable** (Google Fonts) | Уже знаком клиенту по Tilda; нейтральный, чистый, отлично читается на мобильном |
| Декор / заголовки разделов | **Cormorant Garamond** или **Playfair Display** | Тонкий serif, как в каталоге; добавляет «мебельной» классики |
| Спаced caps (РАЗМЕРЫ, МАТЕРИАЛ) | Montserrat 500/600 + `letter-spacing: 0.15em` | Воспроизводит стилистику каталога |

Подключаем через `next/font/google` для предотвращения FOUC (см. [[ADR-002]]).

### Spacing letter

В каталоге активно используется `letter-spacing` (разреженный текст) для подписей —
это часть стилистики. Закладываем utility-класс или mixin:

```scss
@mixin caption-spaced {
  text-transform: uppercase;
  letter-spacing: 0.15em;
  font-weight: 500;
  color: var(--color-text-secondary);
}
```

## Стилистика фотографии

- Светлые интерьеры с большими окнами и мягким светом.
- Кровати в реальных детских комнатах, не на белом cyclorama.
- Мягкие пастельные акценты (текстиль, игрушки, шторы) — но без перегруза.
- Кадр почти всегда фронтальный или со слегка лево/право-смещённого угла.
- Логотип ELIS-MEBEL накладывается на фото поверх в правом-верхнем секторе
  без обводки/подложки.

## Tone of voice (черновик, для воспоминания)

- Тёплый, доверительный, заботливый.
- «Безопасный сон вашего ребёнка» — не «купите кроватку прямо сейчас».
- Акцент на **натуральные материалы**, **гипоаллергенное покрытие**,
  **производство в Беларуси**.
- Без сленга, без эмодзи в основном тексте, без кричащих CTA.

См. отдельный документ [[Voice & tone]] (планируется).
