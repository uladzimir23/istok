# Исток-мебель — Visual language

> **TL;DR.** Головной бренд «Исток-мебель» — B2B/госзаказ. Тон: серьёзный,
> производственный, надёжный. Сейчас брендбука у клиента нет — есть только
> логотип в SVG. Палитра ниже — рабочая, согласовывается с дизайнером в
> процессе Phase 1. ELIS-палитра (см. соседний документ) **не применяется**
> на корпоративном сайте — только в разделе `/krovatki`.

## Источник

- Логотип SVG `Компания Исток - Вектор.svg` — единственный материал по бренду
  у клиента на момент Phase 0.
- Tilda-сайт: использует те же шрифты (Montserrat + Comfortaa), но без выраженной
  фирменной палитры — типичный «магазинный» бежево-белый.

## Палитра (рабочая, проект-эталон)

> Без утверждённого брендбука клиента закладываем **нейтральную деловую B2B-палитру**,
> которую можно безболезненно поменять при появлении дизайнерского решения. Цель —
> чтобы даже без брендинга сайт выглядел профессионально.

| Token | Hex | Назначение |
| --- | --- | --- |
| `--istok-bg-primary` | `#FAFAF7` | Основной фон — почти белый с тёплым подтоном |
| `--istok-bg-section` | `#F1EDE0` | Чередующиеся секции (общий с ELIS — единая родственная теплота) |
| `--istok-text-primary` | `#1F1B17` | Заголовки, body — тёмно-древесный, не чистый чёрный |
| `--istok-text-secondary` | `#5B5046` | Подписи, мета-информация |
| `--istok-text-muted` | `#9C9389` | Tertiary-текст, разделители |
| `--istok-accent-primary` | TBD | Акцент CTA — обсуждается; кандидаты: глубокий бордовый `#7A2E2E` (под театральные кресла) или тёплый бронзовый `#9C6B3A` (под дерево) |
| `--istok-line` | `#E8E2D5` | Тонкие разделители, обводки карточек |

### Принципы

- **Минимум цвета**, акцент один. B2B-аудитория тендерных специалистов и закупщиков
  отвлекается от излишеств; чистая структура и реалистичные фото производства
  работают сильнее, чем «дизайнерский» сайт.
- **Тёплый подтон, не холодный.** Древо — основной материал производства, бежево-кремовый
  фон коммуницирует это лучше, чем синий/серый.
- **Контраст высокий** для читаемости таблиц размеров, технических данных, документации
  по тендерам.

## Типографика (рабочая)

| Слой | Кандидат | Почему |
| --- | --- | --- |
| Заголовки | **Manrope Variable** или **Inter Variable** (Google Fonts) | Современный, чистый, geometric sans — нейтральный для B2B |
| Body | Тот же что заголовки, weight 400 | Согласованность |
| Numerical / specs | Tabular figures у выбранного шрифта | Размеры, веса, размерные сетки в каталоге |

Без поддержки клиента по бренд-гайду используем те же шрифты, что и в reference-проекте
[[../../../../moreminsk]] / [[../../../../comforthotel]] (Manrope Variable + Lora Variable accent),
чтобы не выдумывать шрифтовую систему до утверждения.

## Логотип

- Файл: `tilda-old/istokmebel/images/...` (несколько версий) + клиентский SVG
  `Компания Исток - Вектор.svg`.
- Перенести в `web/public/brand/istok/logo.svg` при имплементации.
- Минимальный размер на мобильном — 120 px по ширине (читаемость текста под лого).

## Чем отличается от ELIS

| Параметр | Исток (B2B) | ELIS (B2C) |
| --- | --- | --- |
| Аудитория | Тендерный специалист, прораб, директор учреждения | Родители, ремонт детской |
| Тон | Серьёзный, производственный | Тёплый, заботливый |
| Палитра | Нейтральная с одним акцентом | Кремово-бежевая, безакцентная |
| Контраст | Высокий, для таблиц | Низкий, для расслабления |
| CTA | «Запросить КП», «Посмотреть портфолио» | «Узнать цену», «Заказать в Telegram» |
| Шрифт акцента | Manrope (sans, geometric) | Cormorant / Playfair (serif, классика) — TBD |
| Фотостиль | Реалистичные снимки производства, объектов госзаказа | Студийные интерьеры детских комнат |

## Что НЕ зафиксировано

- Финальный CTA-цвет (бордовый vs бронзовый) — выбирается с дизайнером.
- Финальные шрифты (Manrope vs Inter, Cormorant vs Playfair) — выбираются с дизайнером.
- Иконография — нет.
- Иллюстрации (если будут) — нет.
- Photography style — без брифа клиента ориентируемся на «реалистичные фото объектов
  и производства», без стилизации.
