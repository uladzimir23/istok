# Исток-мебель — Visual language

> **TL;DR.** Головной бренд «Исток-мебель» — B2B/госзаказ. Тон: серьёзный,
> производственный, надёжный. Палитра извлечена пиксель-пиком из CSS старого
> Tilda-сайта (см. [[../../55 - Design System/Tilda reference]]) — клиент
> хочет сохранить узнаваемость. ELIS-палитра (см. соседний документ)
> **не применяется** на корпоративном сайте — только в разделе `/krovatki`.

## Источник

- **Tilda-сайт** (полный экспорт в `tilda-old/`): 60 CSS-файлов, 25 уникальных
  цветов, реально брендовых — 5–6.
- **Acid test:** `#FF8562` — color для всех `<a>` на сайте (`#allrecords a`,
  `.t-records a`). Это **фактический брендовый акцент**.
- Шрифты: Montserrat (185 деклараций) + Comfortaa (40) — оставляем
  узнаваемость.
- Логотип SVG `Компания Исток - Вектор.svg` — клиент предоставил вектор.

## Палитра

| Token | Hex | Назначение | Источник |
| --- | --- | --- | --- |
| `--color-bg-primary` | `#FFFFFF` | Основной фон | Tilda CSS (243x) |
| `--color-bg-section` | `#F8F8F8` | Чередующиеся секции | Tilda CSS (34x) |
| `--color-text-primary` | `#292929` | Body, заголовки | Tilda CSS (230x) |
| `--color-text-strong` | `#000000` | Контрастные элементы | Tilda CSS (258x) |
| `--color-text-muted` | `#777777` | Подписи, meta | Tilda CSS (41x) |
| `--color-line` | `#BDBDBD` | Тонкие границы | Tilda CSS (33x) |
| `--color-line-soft` | `#DDDDDD` | Разделители | Tilda CSS (44x) |
| **`--color-accent`** | **`#FF8562`** | **Главный акцент (CTA, ссылки, выделения)** | **Tilda CSS (92x)** |
| `--color-accent-hover` | TBD | Hover state | См. [[../../55 - Design System/Design tokens decision]] §1 |

> **Открытая развилка по контрастности:** `#FF8562` на белом даёт
> contrast-ratio ≈ 3.0:1 — проходит AA только для крупного текста.
> Финал решения — в [[../../55 - Design System/Design tokens decision]] §1
> (рекомендация: затемнить до `#E66B47`, ratio 4.6:1, AA для small text).

### Принципы

- **Минимум цвета**, акцент один. B2B-аудитория тендерных специалистов и закупщиков
  отвлекается от излишеств; чистая структура и реалистичные фото производства
  работают сильнее, чем «дизайнерский» сайт.
- **Тёплый подтон, не холодный.** Древо — основной материал производства, бежево-кремовый
  фон коммуницирует это лучше, чем синий/серый.
- **Контраст высокий** для читаемости таблиц размеров, технических данных, документации
  по тендерам.

## Типографика

| Слой | Шрифт | Источник |
| --- | --- | --- |
| Заголовки и body | **Montserrat Variable** (weights 100..900) | Tilda (185 деклараций — доминант) |
| Акцент / декор | **Comfortaa Variable** (weights 300..700) | Tilda (40 деклараций) |

Подключение через `next/font/google` с `subsets: ["latin", "cyrillic"]`,
`display: "swap"` для предотвращения FOUC.

> Текущий placeholder в `web/src/app/layout.tsx` — Manrope. Заменяется на
> Montserrat при имплементации DS (см. [[../../55 - Design System/Design tokens decision]]).

## Логотип

- Файл: `tilda-old/istokmebel/images/...` (несколько версий) + клиентский SVG
  `Компания Исток - Вектор.svg`.
- Перенести в `web/public/brand/istok/logo.svg` при имплементации.
- Минимальный размер на мобильном — 120 px по ширине (читаемость текста под лого).

## Чем отличается от ELIS

| Параметр | Исток (B2B) | ELIS (B2C) |
| --- | --- | --- |
| Аудитория | Тендерный специалист, прораб, директор учреждения | Родители, ремонт детской |
| Тон | Серьёзный, производственный | Тёплый, заботливый |
| Палитра | Нейтральная с одним акцентом | Кремово-бежевая, безакцентная |
| Контраст | Высокий, для таблиц | Низкий, для расслабления |
| CTA | «Запросить КП», «Посмотреть портфолио» | «Узнать цену», «Заказать в Telegram» |
| Шрифт акцента | Manrope (sans, geometric) | Cormorant / Playfair (serif, классика) — TBD |
| Фотостиль | Реалистичные снимки производства, объектов госзаказа | Студийные интерьеры детских комнат |

## Что НЕ зафиксировано

- Точный hover-вариант коралла (`#FA876B` сырой / `#E66B47` доступный / другой) — см.
  [[../../55 - Design System/Design tokens decision]] §1.
- Иконография — lucide-react vs собственный SVG-набор.
- Иллюстрации (если будут) — нет.
- Photography style — без брифа клиента ориентируемся на реалистичные фото
  объектов и производства, без стилизации.
