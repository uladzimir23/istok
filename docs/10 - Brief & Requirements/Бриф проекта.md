# Бриф проекта istokmebel.by

> **TL;DR.** Многопрофильная белорусская мебельная фабрика «Исток-мебель» (Минск + Березино),
> 3 направления производства: театральные кресла (B2B/госзаказ — основной поток), корпусная
> мебель и детские кроватки «Элис» (B2C). Сайт сейчас на Tilda, чисто витринный, без цен,
> заявок и аналитики. Мигрируем на собственный стек, сохраняя бренд-архитектуру: один
> корпоративный сайт `istokmebel.by`, «Элис» — раздел внутри.

## Бренд-архитектура

Двойной бренд, **один сайт**:

1. **«Исток-мебель»** — головной B2B/госзаказ бренд. Корпоративная подача, серьёзность,
   фокус на надёжности, сертификации, гипоаллергенности (для гос-заказа в детские
   учреждения это важно).
2. **«ELIS-MEBEL Kids Beds»** — потребительский суббренд для линейки детских кроваток.
   Эмоциональная подача, фото в интерьере, тёплая палитра (кремовый/бежевый, видно по
   PDF-каталогу).

**Решение 2026-05-01:** «Элис» реализуется как **раздел внутри `istokmebel.by`**,
отдельного домена/сайта НЕТ. Это снижает накладные на инфраструктуру и поддерживает
кросс-продажи (родитель ребёнка может оказаться завхозом школы, и наоборот).

## Три направления производства

### 1. Театральные кресла (B2B / госзаказ) — основной поток

Модели по их каталогу:
- **М1**, **М2** — базовые серии.
- **М3**, **М3-1**, **М3-Г** — основная семья кресел для залов.
- **ПМ-1** — отдельная модификация.

**Целевая аудитория:** тендерные специалисты учреждений культуры, вузов, школ; снабженцы
муниципалитетов; госзаказ.

**Что нужно от клиента:**
- Каталог моделей с размерами, материалами обивки, опциями (откидное сиденье, нумерация,
  складные подлокотники), вариантами тканей.
- Кейсы реализованных проектов: какой объект, год, объём поставки, фото.
- Лицензии, сертификаты, регистрация в реестре отечественных производителей (если есть).

### 2. Корпусная мебель

Подкатегории по сайту: **комоды**, **столы**, **стеллажи**, **шкафы**.

**Целевая аудитория:** B2B (офисы, школы, детские сады), B2C (физлица).

**Что нужно от клиента:**
- Перечень моделей с фото, размерами, материалами.
- Варианты комплектации.
- Прайс или ценовые диапазоны.

| 3 | Паулина | Элис-3 | Односпальная с бортиком | Лесенка-перекладины со стороны изголовья |
| 4 | Эльза | Элис-4 | Односпальная с бортиком | Накладки на спинку — масло либо окраска |

**Унифицированные характеристики:**
- 3 размера спальных мест: 1600×800, 1800×800, 2000×900.
- Материал: натуральная берёза + MDF.
- Покрытие: гипоаллергенный лак (по запросу — масло).
- Опции: выдвижные ящики, бортик-накладка с любой стороны, дополнительное спальное
  место (на ряде моделей).
- Вес: 23 кг (односпальные), 48–50 кг (двухъярусные).

**Целевая аудитория:** родители 2-7-летних детей; родственники, выбирающие подарок;
дизайнеры детских комнат.

**Что нужно от клиента:**
- **Прайс на «Элис»** (8 моделей × 3 размера) — критично для запуска раздела.
- Реальные отзывы, чтобы заменить плейсхолдеры.
- Список магазинов/шоурумов, где можно посмотреть вживую (если есть).
- Условия гарантии и доставки.

## Что НЕ устраивает на текущем сайте (зачем мигрируем)

1. **Нет цен** ни на одной странице.
2. **Нет онлайн-заявки** — клиент уходит звонить.
3. **Нет аналитики** (ни Метрики, ни GA4) — невозможно понять, кто откуда пришёл.
4. **Нет страницы «О компании»** с историей, опытом, сертификатами.
5. **Госзаказ-направление подано слишком минималистично** для тендерного специалиста.
6. **Нет каталога-as-data** — каждое изменение цены/наличия требует ручной правки страниц
   на Tilda.
7. **Нет SEO-инфраструктуры** уровня schema.org Product / Organization / LocalBusiness —
   Yandex и Google не видят сайт как структурированный каталог.

## Цели миграции

1. **Принимать заявки онлайн** — формы → Telegram (Анне) + email бэкапом.
2. **Подключить аналитику** — Яндекс.Метрика + GA4 + Webvisor.
3. **Каталог-as-data** — товары в PocketBase, обновление через админку, сайт
   ревалидируется автоматически.
4. **Сделать B2B-страницы серьёзнее** — отдельный лендинг «Госзаказы» с CTA «Запросить
   КП» и формой под тендерного специалиста.
5. **Сделать раздел «Элис» полноценным** — фильтры по моделям, выбор размера, цвета,
   опций, быстрая заявка через мессенджер.
6. **SEO-инфраструктура** — schema.org, sitemap, hreflang (если будет английская
   версия), OG-картинки автогенерируемые.
7. **Сохранить `istokmebel.by`** — Tilda работает до cutover'а, потом DNS переключается
   на новый сайт.
