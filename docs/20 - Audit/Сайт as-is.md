# Сайт istokmebel.by — аудит as-is (2026-05-01)

> **TL;DR.** Сайт на Tilda, чисто информационный. Нет цен, заявок, корзины, аналитики,
> страницы «О компании», сертификатов. B2B-секция (госзаказ + кресла) подана слишком
> минималистично. Каталог не структурирован как данные. SEO baseline отсутствует.

## Платформа

**Tilda.** Подтверждается DNS / шаблонами. Из этого следует:
- Контент намертво вшит в страницы (нет API/CMS, выгрузка → ручной экспорт).
- Кастомная логика ограничена встроенными блоками + Zero Block.
- Аналитика и пиксели — только то, что встроено в Tilda-настройки (на момент аудита
  не подключены, см. ниже).
- Cutover на новый стек = переключение DNS + редиректы со старых URL.

## Что отсутствует или слабо

### Каталог и продажи

🔴 **Нет цен** ни на одной странице. Невозможно сравнить, сегментировать, посчитать
бюджет.

🔴 **Нет онлайн-заявки** / корзины / формы заказа. Единственный путь конверсии —
позвонить или написать в мессенджер.

🔴 **Нет карточек товаров** с размерами, материалами, опциями. Каталог «Элис» виден
только в PDF-каталоге, который надо качать.

🔴 **Нет фильтров** (по размеру, цвету, типу — односпальная/двухъярусная/чердак,
бюджету).

### B2B / госзаказ

🔴 **Страница `/goszakaz2` минимальна** — нет лицензий, сертификатов, реестра отечественных
производителей, формы запроса КП, кейсов реализованных проектов с указанием объекта/года/
объёма.

🔴 **Каталог театральных кресел** — только имена моделей. Тендерному специалисту нечего
вложить в техзадание.

### О компании

🔴 **Нет страницы «О компании»** с:
- Историей (год основания, ключевые этапы).
- Производственными возможностями (станки, штат, объём в год).
- Сертификатами (гипоаллергенность, госстандарт, реестр производителей).
- Кейсами и портфолио с атрибуцией («Кресла для НГТУ, 2023, 450 мест»).

### Аналитика и маркетинг

🔴 **Ни Яндекс.Метрики, ни GA4** на странице. Подтверждено через DevTools / view-source —
ни одного аналитического скрипта.

🔴 **Нет пикселей** (Meta, ВК, MyTarget). Если когда-то была реклама — невозможно
ретаргетить.

🔴 **Нет UTM-разметки** на исходящих ссылках в соцсети / партнёров.

### SEO

🔴 **Нет structured data** (schema.org) — ни Product, ни Organization, ни LocalBusiness
(критично для Минска), ни ItemList для категорий.

🟡 **Sitemap и robots** — Tilda что-то генерирует автоматически, но не проверено.

🟡 **OG-картинки** — Tilda ставит дефолтные, не уникальные на товар/раздел.

🟡 **hreflang** — нет (если будет английская версия — критично; если только русский — ок).

### UX / доступность

🟡 **Мобильная адаптация** — Tilda даёт базовую, но не контролируемую (адаптация под
конкретный товарный UX невозможна без покупки про-блоков).

🟡 **Не проверено**: skip-link, контрастность, alt-теги изображений.

### Технические

🟡 **Производительность** — Tilda исторически грешит избыточными скриптами и шрифтами.
Lighthouse не замерён, но визуально загрузка > 2 секунд на 3G.

🟡 **Скорость отдачи изображений** — Tilda CDN ок, но без AVIF/современных форматов
оптимизации.
