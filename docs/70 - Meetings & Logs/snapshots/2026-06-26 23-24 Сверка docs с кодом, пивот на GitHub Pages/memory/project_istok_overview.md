**Бренд-архитектура двойная (но один сайт):**
- «Исток-мебель» — головной B2B/госзаказ бренд (corporate site).
- «ELIS-MEBEL Kids Beds» — потребительский суббренд линейки детских кроваток. **Отдельного сайта НЕТ.** «Элис» — это раздел/линейка внутри `istokmebel.by`. Решено 2026-05-01.

**Три направления производства (отражены в навигации):**
1. Театральные кресла — модели М1, М2, М3, М3-1, М3-Г, ПМ-1. По их заявлению ~1/3 госзаказа на кресла для залов в РБ. Основной денежный поток.
2. Корпусная мебель: комоды, столы, стеллажи, шкафы.
3. Детские кроватки «Элис» — 8 моделей: Николь (Элис-1), Мелита (Элис-2, двухъярусная), Паулина (Элис-3), Эльза (Элис-4), Доминика (Элис-5), Афина (Элис-6), Ясмина (Элис-7), Мишель (Элис-8, двухъярусная). Все с 3 размерами спальных мест: 1600×800, 1800×800, 2000×900. Материал: берёза + MDF, гипоаллергенный лак. Вес: 23 кг (одно), 48–50 кг (двух).
