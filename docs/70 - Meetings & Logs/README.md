# 70 — Meetings & Logs

Хроника проекта.

## Содержимое

- Session logs — итоги сессий с Claude Code (формат `YYYY-MM-DD HH-mm <topic>.md`,
  см. `.claude/skills/session-log.md`).
- Встречи — расшифровки/итоги встреч и созвонов с клиентом.
- `snapshots/` — frozen-копии Dashboard + dept-lenses + memory + git-log на момент
  каждого session log.

## Правила

- **Session log** обязателен после содержательной сессии (с реальными изменениями).
- **Snapshot** делается сразу за session log в `snapshots/<тот же slug>/`.
- Шаблон — `docs/80 - Templates/session-log.md`.
- Полная инструкция — `.claude/skills/session-log.md` + `.claude/skills/snapshot.md`.
