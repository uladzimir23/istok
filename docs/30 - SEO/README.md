# 30 — SEO

SEO-стратегия и операционные документы. Заполняется после Phase 1 (когда зафиксирован
стек и понятна структура страниц).

## Что сюда пойдёт

- Стратегия SEO тремя волнами (baseline, контент, авторитет).
- Словарь meta (title/description/canonical паттерны по типам страниц).
- Карта schema.org (Product, ItemList, Organization, LocalBusiness, BreadcrumbList).
- Список ключевых запросов по направлениям (B2B-кресла, госзаказ, корпусная, детские
  кроватки «Элис»).
- Хвост запросов по моделям «Элис».
