# 80 — Templates

Шаблоны для новых документов. Копируем и заполняем.

## Содержимое

- `adr.md` — шаблон Architecture Decision Record.
- `session-log.md` — шаблон session log от Claude Code.
- `dept-dashboard.md` — шаблон новой dept-линзы (например, для будущих отделов
  «Производство» или «Финансы»).

## Связанное

- `.claude/skills/docs-writing.md` — общий формат документов.
- `.claude/skills/multi-audience-docs.md` — паттерн dept-линз.
- `.claude/agents/adr-drafter.md` — sub-агент, заполняющий ADR из обсуждения.
