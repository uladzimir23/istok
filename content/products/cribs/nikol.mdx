# Николь (Элис-1)

Детская кроватка «Николь» из натуральной берёзы. Гипоаллергенное лакокрасочное покрытие. 3 размера(ов) спального места.

## Особенности

- Борт устанавливается с любой стороны
- Выдвижные ящики
- Дополнительное выдвижное спальное место
- Бортик-накладка
- Возможность трансформации в двухъярусную / кровать-чердак
